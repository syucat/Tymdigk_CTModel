# Tymdigk_CTModel

Cell Tracking Challenge の `Fluo-N3DH-SIM+` データを使い、U-Netで細胞セグメンテーション
(各ピクセルが細胞か背景か)を学習し、Zスライス間でインスタンス追跡(タグ付け)まで行う実験一式です。

学習済みモデルと生成済みの可視化結果(`20260708/results/result.html`)を含んでいるので、
学習をゼロから回さなくても結果をすぐ確認できます。

このリポジトリは配布・共有用に、研究用リポジトリ `Tymdigk_Codes` から本実験
(`20260708/`)だけを切り出したものです。`Tymdigk_Codes`とは別物・無関係の
リポジトリで、`Tymdigk_Codes`側を参照・依存することはありません。

## はじめに

**`20260708_INTRO/COMMANDS.md`を読んでください。** セットアップ手順・
「何がしたいか」ごとの実行コマンド・注意点をまとめています。

## 構成

```
20260708/        本体(学習・推論・可視化のPythonスクリプト、学習済みモデル、生成済み結果)
20260708_INTRO/  セットアップ手順・コマンド集(COMMANDS.md)・requirements.txt

```
